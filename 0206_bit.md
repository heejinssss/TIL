# 비트 연산으로 부분집합 구하기

### 준비물

```python
arr = ['Fish', 'And', 'Chips']
N = len(arr)

# arr에 대한 모든 경우의 수

for i in range(1 << N):
    # j : arr의 idx
    for j in range(N):
        if i & (l << j):
            print(arr[j], end=' ')
    print()
```

### 실행 결과

```python
Fish 
And 
Fish And 
Chips 
Fish Chips 
And Chips 
Fish And Chips
```

### `1 << 2`의 의미 : 피연산자를 비트열에서 왼쪽으로 이동시킨다.

```python
N = 3
print(1 << N) # 8
```

| 2진수 | 10진수 | shift 횟수 |
| --- | --- | --- |
| 0001 | 1 | 0 |
| 0010 | 2 | 1 |
| 0100 | 4 | 2 |
| 1000 | 8 | 3 |
- `for i in range(1 << N)`
    - `for i in range(8)`
    - 3개의 원소로 만들 수 있는 모든 경우의 수는 8가지 (2^3)

- `arr = ['Fish', 'And', 'Chips']`의 각 요소를 비트로 나타내면
    - 1, 2, 3번째 요소에 해당
    - arr의 index를 경우의 수에 따라 부분집합에 포함을 시킬 것인가, 말 것인가

| Fish | And | Chips |
| --- | --- | --- |
| 001 | 010 | 100 |

- `if i & (1 << N)`
- 이 구문을 통과하기 위해서는 i값의 오른쪽부터 n번째 비트 값이 1이어야
- 비트 연산자 & (and) 두 비교 대상의 비트 값의 각 자리 수를 비교
- 양쪽의 자리 수가 모두 True(1)일 때, True를 반환
- 숫자 3(N)을 받고 i(3)번째 경우의 수
- `3 & (1 >> 0)` → True

| 3 | 0 | 1 | 1 |
| --- | --- | --- | --- |
| fish | 0 | 0 | 1 |
| result | 0 | 0 | 1 |
- `3 & (1 << 1)` → True

| 3 | 0 | 1 | 1 |
| --- | --- | --- | --- |
| and | 0 | 1 | 0 |
| result | 0 | 1 | 0 |
- `3 & (1 << 2)` → False

| 3 | 0 | 1 | 1 |
| --- | --- | --- | --- |
| chips | 1 | 0 | 1 |
| result | 0 | 0 | 0 |
- 즉, 배열 `arr = ['Fish', 'And', 'Chips']`의 부분집합의 경우의 수 중
- 3번째 경우는 `Fish, And` 두 요소를 포함한 것과 같다
